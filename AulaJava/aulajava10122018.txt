JPA:Java Persistence API
- Para salvar no banco de dados precisa ser Enitidades. declarar o @Entity no código.
@Column para criar colunas ex: @Column(name = "complemento", length =20, nullable = false)
-Para Gerenciar os banco de dados usa o Entity Manager -> abre transações, salva entidades no banco, cria banco de dados de forma automática.
- para usar o Entity manager precisa do entity manager Factory.
- persistence XML.

windows: usa o docker toolbox para fazer conexão virtualmente com banco de dados sem instalar o programa.
